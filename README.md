# Калькулятор переплавки сплавов (TFC)

Веб-калькулятор для TerraFirmaCraft. Помогает рассчитывать плавку пыли в слитки и составление сплавов по рецепту.

## Возможности

### Плавка
- Расчёт общего объёма металла из пыли (мб)
- Количество целых слитков и остаток
- Минимальное количество пыли для ровного числа слитков
- Расчёт необходимых сосудов

### Сплавы
- Задание рецепта: материалы, проценты (фикс или диапазон), мб/ед
- Два режима:
  - **Суммирование** — если указано несколько материалов, просто складывает
  - **Подбор** — если указан один материал или цель в слитках, подбирает остальные с минимальным отклонением от рецепта
- Сохранение рецептов в браузере (localStorage)
- Импорт/экспорт JSON
- Галерея сохранённых рецептов

## Запуск

Просто открой `index.html` в браузере. Никаких зависимостей.

## Технологии

Чистый HTML, CSS, JavaScript. Никаких фреймворков.
