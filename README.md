# Калькулятор переплавки сплавов (TFC)

## Плавка

Считает объём металла из пыли, целые слитки, остаток, минимальное количество пыли для ровного числа слитков, количество сосудов.

## Сплавы

Два режима: **суммирование** (несколько материалов — просто сложить) и **подбор** (один материал или цель в слитках — подбирает остальные с минимальным отклонением от рецепта). Проценты можно задавать фиксированным числом или диапазоном.

Рецепты сохраняются в браузере, импорт/экспорт JSON.
