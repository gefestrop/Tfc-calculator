# ⚙️ Калькулятор переплавки (TFC)

Инструмент для **TerraFirmaCraft**: плавка пыли → слитки и расчёт сплавов по рецепту.

---

## 🔥 Плавка

Вводишь количество пыли, мб/пыль, мб/слиток — получаешь:

- объём металла (мб)
- целые слитки и остаток
- минимальное количество пыли для ровного числа слитков
- количество сосудов

```
Пыль: 48 шт × 129 мб/ед → 6192 мб → 43 слитка (432 мб остаток) → 3 сосуда
```

---

## 🧪 Сплавы

Указываешь материалы, проценты (фикс или диапазон), мб/ед — и выбираешь режим.

### 📥 Суммирование

Просто складывает то, что ввёл. Показывает общий объём и слитки.

> Медь 15 шт × 129 + Висмут 6 шт × 144 + Цинк 9 шт × 129 = **3960 мб** / 144 = **27 слитков**

### 🎯 Подбор

Вводишь **один материал** или **цель в слитках** — калькулятор подбирает остальные с минимальным отклонением от рецепта.

> Цель: **8 слитков** (1152 мб) · Рецепт: Сталь 25%, Розовое золото 12.5%, Латунь 12.5%, Чёрная сталь 50%  
> → Сталь **2 шт**, Роз.золото **1 шт**, Латунь **1 шт**, Чёрн.сталь **4 шт**

### 💾 Рецепты

Сохраняются в браузере. Можно экспортировать/импортировать JSON.
